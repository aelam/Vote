UMSocialControllerService Class Reference

Inherits from NSObject
Declared in UMSocialControllerService.h
UMSocialControllerService.m

Overview

用此类的方法可以得到分享的有关UI对象，例如分享列表、评论列表、分享编辑页、分享授权页、用户中心页面等。返回都是UINavigationController对象，建议把这个对象present到你要添加到的UIViewController上

  • – getSocialAccountController

    个人中心页面，该页面包括个人的各个微博授权信息
  • – getSocialShareEditController:

    分享编辑页面
  • – getSocialOauthController:

    授权页面，如果你要想得到授权完成之后的事件，你可以实现UMSocialUIDelegate里面的-(void)didCloseUIViewController:(UMSViewControllerType)fromViewControllerType;方法，当授权关闭页面会调用此方法。另外授权完成之后sdk会自动去取个人账户信息，你可以在回调函数里面去到刚刚授权的微博平台的账户信息。
  • – getSocialLoginController

    登录页面,出现你配置出现的所有sns平台，授权之后设置为sdk的登录账号。使用评论功能时会取此登录账号的昵称和头像。
  • + setSocialConfigDelegate:

    设置实现了UMSocialConfigDelegate的对象，类方法，表示该对象对全部UMSocialControllerService对象起作用
